DICIONÁRIO DE FLUXO E AÇÕES DO SISTEMA (SPA)
TELA 01: LOGIN (PONTO DE ENTRADA)
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
ENTRAR Usuário, Senha Valida credenciais no AuthService. Se correto, redireciona para a Home.
TELA 02: HOME (MENU PRINCIPAL)
ELEMENTO/FUNÇÃO INTERAÇÃO RESULTADO / FLUXO
Tiles de Navegação Clique em qualquer Tile (ex: ✈️) Redireciona para o componente SPA correspondente no Main Content.
Sidebar (Menu) Seleção de item Navegação lateral rápida entre módulos ativos.
LOGOUT Clique (Header) Encerra sessão no AuthService e retorna à tela de Login.
TELA 03: GESTÃO DE AERONAVES
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
Listagem N/A (Automático) Carrega todas as aeronaves via AeronaveService.listarTodas().
SALVAR (Cadastro) Cód, Modelo, Tipo Persiste nova aeronave no AeronaveRepository. Atualiza a lista acima.
TELA 04: INVENTÁRIO DE PEÇAS
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
Estoque Geral N/A (Automático) Carrega lista de peças via PecaService.listarTodas().
VINCULAR Seleção Peça, Cód Aero Associa a peça física a uma unidade via AeronaveService.adicionarPeca().
TELA 05: LINHA DE PRODUÇÃO (KANBAN)
ELEMENTO/FUNÇÃO INTERAÇÃO RESULTADO / FLUXO
Visualização Kanban N/A (Monitoramento) Exibe aeronaves divididas por StatusEtapa (Pendente/Execução/Concluído).
Fluxo Lógico: O sistema impede o avanço de aeronaves (via ProducaoService) sem a conclusão da etapa anterior.
TELA 06: CONTROLE DE QUALIDADE
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
Histórico N/A (Automático) Lista resultados de TipoTeste para cada aeronave.
SALVAR (Teste) Aero, Tipo, Resultado Registra aprovação/reprovação no TesteService. Essencial para o relatório final.
TELA 07: RELATÓRIOS
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
GERAR ARQUIVO .TXT Aero, Nome Cliente Compila dados via RelatorioService. Exporta arquivo físico para a pasta /relatorios.
TELA 08: GESTÃO DE EQUIPE
ELEMENTO/FUNÇÃO ENTRADA (INPUT) RESULTADO / FLUXO
Listagem Equipe N/A (Automático) Exibe todos os funcionários e seus níveis (FuncionarioService).
CADASTRAR Dados Cadastrais Cria novo usuário com permissões específicas (Admin/Eng/Op).
EDITAR COLABORADOR Seleção de ID Abre campos para modificação de dados de um funcionário existente.
